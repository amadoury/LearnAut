## Test 
Pour tester, il faut écrire dans fichier `data.txt` dans l'ordre : 
- les mots acceptés (un mot par ligne)
- un trait `-`
- les mots non acceptés

## Compilation et Execution
Ouvrir un terminal et se placer dans le répertoire automata
- aalpy : 
`make py`
- LearnLib :
`make java`
- les deux :
`make both`
